# 📋 Clipboard-AI

**Помощник для работы с буфером обмена через NVIDIA AI**

Автоматически обрабатывает скопированный текст (особенно C# код), отправляет в модель NVIDIA, очищает от markdown-блоков и открывает результат в notepad.

---

## ✨ Возможности

- **Скрытый режим** — работает в фоне без окон
- Автоматическая **очистка** markdown-кода (```csharp, ```c# и т.д.)
- Автоматическое открытие результата в **Notepad** (с расширением `.cs`)
- Горячие клавиши для удобной работы
- Специально оптимизирован для **C#**
- Поддержка NVIDIA NIM
---

## 🚀 Как использовать

1. Скопируй любой текст или код (`Ctrl + C`)
2. Нажми **Ctrl + D** — запрос отправится в AI
3. Через пару секунд автоматически откроется **блокнот** с чистым результатом

**Горячие клавиши:**
- `Ctrl + D` — обработать буфер обмена
- `Ctrl + Q` — завершить программу
